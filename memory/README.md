# Memory Bank

This directory stores persistent project context for future AI sessions.

Canonical files:

- `project-context.md`: stable project facts and conventions
- `current-context.md`: current focus, active UI state, short-term notes
- `change-log.md`: concise log of significant completed changes

Usage:

- Read the canonical files at the start of a session.
- Update them after significant changes.
- Keep notes brief and practical.
